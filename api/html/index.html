Aircraft Database API

This is the API for the aircraft database project. In order to interact with it there are a few key functions:

  1. newMessage: Adds a new aircraft message to the database
  2. addNote: Adds a description to an exsisting aircraft which is in the database
  3. **getAircraft: Get information on an aircraft** not implemented
  4. **getFlight: Get information on a specific flight of an aircraft** not implemented
  5. setStation: Set a new base station
  6. getStations: Get information on a base station
