Aircraft Database API

This is the API for the aircraft database project. In order to interact with it there are a few key functions:

  • newMessage: Adds a new aircraft message to the database
  • addNote: Adds a description to an exsisting aircraft which is in the database
  • setStation: Set a new base station
  • getStations: Get information on all base stations
  • lastTimeFromStation: Get the last message timestamp recieved from a station
  • crossRef: Cross reference an ICAO hex or callsign
  • getAircraft: Get information about an aircraft
  • getFlights: Get a list of flights from a single aircraft
  • getRecentFlights: Get a list of recently seen flights, based on a specified number of minutes in the past
