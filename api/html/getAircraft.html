Get List of attributes about an aircraft

This function will get a list of attributes about an aircraft, results are returned as a json format with average altitude, total flights seen, latest station which saw this plane, average speed, last seen at time, last position, last flight number, and some user notes

API Arguments

  • "id" (Required): Aircraft ICAO identifier, 21 bit hex code

Example query: /getAircraft?id=C06359 (Must be a GET HTTP request)
